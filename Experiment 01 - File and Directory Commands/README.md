## Objective
To study basic Linux file and directory management commands.

## Commands Used
`pwd`, `ls`, `cd`, `mkdir`, `rmdir`, `rm`, `cp`, `mv`, `cat`, `touch`

## Procedure
1. Open the terminal.
2. Use `pwd` to display the current directory.
3. Create a new directory using `mkdir test`.
4. Navigate inside it using `cd test`.
5. Create a file using `touch file1.txt`.
6. List files using `ls`.
7. Copy, move, and delete files to test each command.
